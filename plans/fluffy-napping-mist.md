## 実装方針

### カラーパレット
| 変数 | 値 | 用途 |
|------|-----|------|
| `--bg` | `#06060f` | 背景（極濃紺黒） |
| `--bg-alt` | `#0d0d1a` | セクション背景 |
| `--surface` | `#131326` | カード背景 |
| `--border` | `rgba(99,102,241,.25)` | ボーダー（紫がかり） |
| `--accent` | `#818cf8` | メインアクセント（インディゴ） |
| `--accent-glow` | `#6366f1` | グロー用 |
| `--cyan` | `#22d3ee` | サブアクセント（シアン） |
| `--text` | `#f1f5f9` | 本文テキスト |
| `--muted` | `#94a3b8` | サブテキスト |

### セクション構成（現行維持）
1. **Nav** — ダーク半透明ガラス + アクセントロゴ
2. **Hero** — 背景にドットグリッドパターン + グロー光源 + メインコピー + CTAボタン
3. **Features** — 3カード（グラデーションボーダー + グロー hover）
4. **CTA** — グラデーション枠付きセクション
5. **Footer** — 極暗背景

### 追加表現
- Hero背景: `radial-gradient` によるグロー光源 + SVGドットパターン
- ボタン: グラデーション塗り + `box-shadow` グロー
- カード: `border: 1px solid var(--border)` + hover時にシアングロー
- アニメーション: 既存の `IntersectionObserver` フェードイン継続使用

## 実装手順

1. `index.html` を Write ツールで完全上書き（CSS変数・HTML構造・スタイル全て更新）
2. ブラウザで動作確認

- `index.html` をブラウザで直接開き、以下を確認

  - スクロール時フェードインアニメーション
  - ボタンホバー時のグロー効果
  - モバイル幅（375px）でのレイアウト崩れなし
